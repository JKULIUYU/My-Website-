# 项目更新日志 (Changelog)
该文件记录由 jkuliuyu 开发的个人全栈网站 (Vite + Vue3 + Tailwind CSS v4) 的版本更新与迭代过程。采用纯净 macOS 莫兰迪极简风格。

## [未发布] - 2026-03-17
### 🌟 新增 (Added)
- 新增「今日代码短句」卡片：每日按日期随机刷新一句与代码相关的幽默短句。
- 点赞功能支持服务端接口与设备限制：对接 `/api/like`，记录总数并限制同设备重复点赞。

### 🛠 修复与优化 (Fixed & Changed)
- 左侧导航不再默认高亮「近期进展」：所有导航项默认统一样式，仅在 hover 时出现文字强调动画。
- 点赞按钮移至主题切换器右侧，更贴近全局控制区。
- 增加页面历史记录支持：使用浏览器返回/前进键可在站内页面切换。
- 所有卡片增加轻微弹跳动画：增强 hover 反馈与灵动感。
- 开发状态提示居中对齐：绿点与文字整体居中显示。
- 卡片弹跳动画更克制：降低位移与缩放，提升高级感。
- 修复构建报错：移除未使用的 `isActive` 定义，避免 TS6133。
- 深色模式文本对比优化：主题切换选中状态文字颜色加深，提升可读性。

## [未发布] - 2026-03-15
### 🌟 新增 (Added)
- 新增页面左下角的 **全自动主题切换控制组 (Theme Switcher)**：
  - 支持 **亮色模式 (Light)**：高级莫兰迪白绿渐变。
  - 支持 **暗色模式 (Dark)**：深邃的极客夜间环境。
  - 支持 **自动模式 (Auto)**：精准基于时间的日夜切换（06:00-18:00 为亮色，其余时间为暗色），实现“日出而作，日落而息”的硬核逻辑。

### 🛠 修复与优化 (Fixed & Changed)
- **修复时钟不对齐问题**：重构了仪表盘的卡片内部 Flexbox 参数，彻底解决大数字时间显示视觉偏上的 Bug。
- **架构记录**：引入了统一的 Changelog 管理范式，方便追踪每一次网站架构调整与 UI 重构。
