# 项目更新日志 (Changelog)
该文件记录由 jkuliuyu 开发的个人全栈网站 (Vite + Vue3 + Tailwind CSS v4) 的版本更新与迭代过程。采用纯净 macOS 莫兰迪极简风格。

## [未发布] - 2026-03-17
### 🌟 新增 (Added)
- 新增「今日代码短句」卡片：每日按日期随机刷新一句与代码相关的幽默短句。
- 点赞功能支持服务端接口与设备限制：对接 `/api/like`，记录总数并限制同设备重复点赞。
- 新增项目详情页：阿克曼转向小车，补充系统架构与控制算法说明。
- 项目列表 P2 卡片可点击：进入 `#/projects/project_ackermann` 独立详情页。

### 🛠 修复与优化 (Fixed & Changed)
- 左侧导航不再默认高亮「近期进展」：所有导航项默认统一样式，仅在 hover 时出现文字强调动画。
- 点赞按钮移至主题切换器右侧，更贴近全局控制区。
- 增加页面历史记录支持：使用浏览器返回/前进键可在站内页面切换。
- 所有卡片增加轻微弹跳动画：增强 hover 反馈与灵动感。
- 开发状态提示居中对齐：绿点与文字整体居中显示。
- 卡片弹跳动画更克制：降低位移与缩放，提升高级感。
- 修复构建报错：移除未使用的 `isActive` 定义，避免 TS6133。
- 深色模式文本对比优化：主题切换选中状态文字颜色加深，提升可读性。
- 深色模式选中按钮文字改为深色：与浅色按钮背景形成更强对比。
- 项目页补充公司项目说明：教育机器人电机控制标注为私有项目，并说明本人负责 PID 模块。
- 项目页补充抽象策略：多圈脉冲累计、动态补偿、校准思路与滤波限幅等。
- 新增项目详情页：教育机器人电机控制，包含背景/角色/策略/应用与算法架构示意图。
- 子页面允许滚动：解决详情页内容超出时被裁切的问题。
- 子页面内容立即渲染：移除内层过渡，避免首次进入空白。
- 页面切换动效升级：加入轻微位移、缩放与模糊过渡，切换更高级。
- 页面切换动效微调：缩短模糊时长并减小模糊强度，避免拖沓感。
- 项目卡片点击反馈回归统一：与全站卡片交互一致。
- 项目卡片进入详情过渡优化：点击后短暂保留卡片抬升效果，再进入详情。
- 项目详情路由调整：支持 `#/projects/project_motor` 形式，并返回到项目列表。
- 项目卡片进入详情延时加长：更容易感知与其他卡片一致的抬升动画。
- 项目卡片进入详情触发方式优化：改为按下即触发抬升与切页，避免动画被吞掉。
- 项目卡片点击跳转修复：改为按钮触发，避免锚点返回首页。
- 项目卡片点击脉冲反馈：保证点击时也能看到与其他卡片一致的动画手感。
- 返回逻辑调整：优先回退浏览器历史的上一页。
- 返回按钮行为修正：优先返回浏览器上一页，无历史则回主页。
- 项目卡片点击兼容性修复：增加 pointer/click 触发，避免部分环境点击无响应。
- 每日短句改为 0 点刷新：按天循环且避免与昨天重复。

## [未发布] - 2026-03-15
### 🌟 新增 (Added)
- 新增页面左下角的 **全自动主题切换控制组 (Theme Switcher)**：
  - 支持 **亮色模式 (Light)**：高级莫兰迪白绿渐变。
  - 支持 **暗色模式 (Dark)**：深邃的极客夜间环境。
  - 支持 **自动模式 (Auto)**：精准基于时间的日夜切换（06:00-18:00 为亮色，其余时间为暗色），实现“日出而作，日落而息”的硬核逻辑。

### 🛠 修复与优化 (Fixed & Changed)
- **修复时钟不对齐问题**：重构了仪表盘的卡片内部 Flexbox 参数，彻底解决大数字时间显示视觉偏上的 Bug。
- **架构记录**：引入了统一的 Changelog 管理范式，方便追踪每一次网站架构调整与 UI 重构。
